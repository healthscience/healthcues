# healthCues Android (Kotlin)

Displays BentoBox logo above the text:

healthCues on BentoBoxDS

To replace the placeholder vector with the real logo, drop a PNG/SVG as:
- app/src/main/res/drawable/bentobox_logo.png (or keep XML vector), then rebuild.

## Run (Android Studio)
Open `healthcues/android`, let Gradle sync, select device, Run.
